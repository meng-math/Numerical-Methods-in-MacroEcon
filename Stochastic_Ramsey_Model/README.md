# Stochastic Ramsey Project

# Author: meng-math

Description: This repisotory includes the numerical simulation of Ramsey model, mainly including：

- discretiazation of AR(1), 
- deteministic VFI with Howard improvement, 
- EGM method for stochastic model，
- stationary distribution simulation.


Key files

- main_stochastic_ramsey.py — orchestration script for solving, simulating and plotting results.

- Ramsey_VFI_Howard_Improvement.py — deterministic VFI with Howard improvement.

- random_Ramsey_EGM.py — stochastic Ramsey solver using EGM.

- tauchen_rouwenhorst.py — Tauchen and Rouwenhorst discretization routines and Markov simulation

- stationary_distribution.py — routines to compute stationary joint and marginal distributions.
